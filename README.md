# My MTG Commander Project

## Setup
1. Flask - venv
    * `cd backend/`
    * `source venv/bin/activate`
    * `python app.py`

2. Tailwind - CSS
    * `cd frontend/`
    * `npm install`
    * `npm run build:css`
    * `npx tailwindcss -i ./src/input.css -o ./static/css/style.css --minify` (For deployment, generates css & removes whitespaces)
